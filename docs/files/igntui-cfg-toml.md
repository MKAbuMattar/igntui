# .igntui.cfg.toml

## NAME

`.igntui.cfg.toml` — per-project sidecar pinning the templates that produced
the `.gitignore`

## LOCATION

The directory containing the generated `.gitignore`. The TUI looks for the
file in the **current working directory** at launch.

```
my-project/
├── .gitignore          # generated content (with managed-block markers)
└── .igntui.cfg.toml     # this file
```

## SYNOPSIS

```toml
schema_version = 1
generated_at   = "2026-04-27T12:00:00Z"
igntui_version = "0.0.2"

[selection]
templates   = ["python", "node", "macos"]
search_mode = "fuzzy"

[output]
path                = ".gitignore"
preserve_user_edits = true
```

## DESCRIPTION

The sidecar records the inputs that produced the neighboring `.gitignore`
so a teammate (or future-you) can reproduce or extend the selection without
re-picking templates. It is intentionally small, human-readable, and
**safe to commit** to git.

When `igntui` (TUI mode) launches, it scans the working directory for this
file. If present and valid, it:

1. Pre-populates `state.selected_templates` from `[selection].templates`.
2. Restores `[selection].search_mode` if recognized.
3. Triggers an immediate content generation.
4. Posts a status message: `✓ Loaded N templates from .igntui.cfg.toml — Press 'c' to clear`.

The sidecar is **rewritten on every save** to capture the latest selection.

## SCHEMA

### Top-level

| Field            | Type    | Required | Description                                |
| ---------------- | ------- | -------- | ------------------------------------------ |
| `schema_version` | integer | yes      | Forward-compat marker. Current value: `1`. |
| `generated_at`   | string  | yes      | ISO-8601 UTC timestamp of last write.      |
| `igntui_version` | string  | yes      | Version of igntui that wrote the file.     |

A `schema_version` newer than this igntui supports triggers a warning but
does not abort load (best-effort tolerance).

### `[selection]` table

| Field         | Type          | Default   | Description                                               |
| ------------- | ------------- | --------- | --------------------------------------------------------- |
| `templates`   | array<string> | `[]`      | Template names to load. Order is preserved.               |
| `search_mode` | string        | `"fuzzy"` | One of `fuzzy`, `exact`, `regex`. Restored on TUI launch. |

If `templates` is not a list, it is silently treated as `[]` and a warning
is logged.

### `[output]` table

| Field                 | Type    | Default        | Description                                                                                                            |
| --------------------- | ------- | -------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `path`                | string  | `".gitignore"` | Where the generated content was written, relative to the sidecar.                                                      |
| `preserve_user_edits` | boolean | `true`         | If true, re-saves keep content outside the managed-block markers. See [Managed blocks](../concepts/managed-blocks.md). |

## EXAMPLES

### Minimal valid file

```toml
schema_version = 1
generated_at   = "2026-04-27T12:00:00Z"
igntui_version = "0.0.2"

[selection]
templates = ["python"]
```

### Full file

```toml
# .igntui.cfg.toml — generated by igntui; safe to commit.
# Re-running igntui in this directory restores this selection.
schema_version = 1
generated_at   = "2026-04-27T12:00:00Z"
igntui_version = "0.0.2"

[selection]
templates   = ["python", "node", "macos"]
search_mode = "fuzzy"

[output]
path                = ".gitignore"
preserve_user_edits = true
```

## OPT-OUT

To skip writing the sidecar from a CLI invocation:

```
igntui generate python --output .gitignore --no-sidecar
```

The TUI always writes the sidecar on save when content is non-empty.

## ERROR HANDLING

| Scenario                           | Behavior                                        |
| ---------------------------------- | ----------------------------------------------- |
| File missing                       | `find_sidecar` returns `None`, TUI starts empty |
| File exists but invalid TOML       | Warning logged, treated as missing              |
| `schema_version` newer than known  | Warning logged, best-effort load                |
| `[selection].templates` not a list | Warning logged, treated as empty list           |
